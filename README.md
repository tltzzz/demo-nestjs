## Описание

Demo NestJS

## Установка

```bash
# node modules
$ npm install

# init app
$ npm run init
```

## Команды

```bash
# docker
$ npm run docker:run

# migration
$ npm run typeorm:run

# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Адреса

Server - http://localhost:3000
PgAdmin - http://localhost:8080
